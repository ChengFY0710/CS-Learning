# 11 - 3 分配内存

malloc函数和free函数都在``stdlib.h``头文件中。

## 01 malloc()函数
```c
malloc(所需的内存字节数);
```
该函数接受一个参数：所需的内存字节数。malloc函数会找到合适的空闲内存块，这样的内存是**匿名**的。即**malloc()分配内存，但是不会为其赋名**。然而，**它确实返回动态分配内存块的首字节地址**。因此，可以将该地址赋值给一个指针变量，并用指针访问该内存。特别地，如果malloc()分配内存失败，将返回**空指针**。

### 通用指针 - 指向void的指针
从ANSI C标准开始，C使用一个新的类型：**指向void的指针**。该类型相当于一个**通用指针**，不指向数据类型，但会存放地址。

```c
double * ptr;
ptr = (double *)malloc(30 * sizeof(double));
```
(double *)为强制类型转换，将malloc返回的void空指针，强制转换成**指向double类型的指针**。

该操作**为30个double类型的值请求内存空间**，并设置ptr指向该位置。相当于```double ptr[30];```，因此``ptr[0]``访问该块的首元素，``ptr[1]``访问该块的第2个元素。

### 创建动态数组
#### （1）声明变长数组
```c
int n;
scanf("%d, &n");
int num[n];
```
用变量表达式表示数组的维度，用数组名访问数组的元素。具有这种特性的数组只能在**自动内存**中创建。
#### （2）调用malloc()函数
```c
int n;
scanf("%d", &n);
int * ptr;
ptr = (int *)malloc(n*sizeof(int)); //相当于int ptr[n];
```
该操作可以是**静态的**也可以是**动态的**。

## 02 free()函数
```c
free(malloc函数之前返回的地址);
```
free()函数的参数是malloc函数之前返回的地址，作用是**释放之前malloc()分配的内存**。因此动态分配内存的存储期是**从调用malloc()函数开始，到调用free()释放内存为止**。

## 03 calloc()函数
```c
int n;
scanf("%d", &n);
int * ptr;
ptr = (int *)calloc(n, sizeof(int));
```
整体的核心逻辑与malloc基本一致。

但区别是：
+ malloc分配的内存**未初始化**。
+ calloc分配的内存被**全部初始化为0**。
