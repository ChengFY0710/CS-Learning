# 11 - 2 常见五种存储类别

## 01 自动变量
|存储类别|存储期|作用域|链接|声明方式|
|---|---|---|---|---|
|自动|自动|块|无|块内|

为了更清楚的表明你的意图，可以使用关键字auto。
```c
int main()
{
    auto int plox;
}
```
关键字auto是**存储类别说明符**。
变量具有自动存储期意味着，程序在进入改变量声明所在的块时，变量创建并存在，程序在退出该块时变量消失。**原来该变量占用的内存位置现在可作他用**。

还有一种：嵌套块的情况。**块中声明的变量仅限于该块及包含的块使用**。
```c
int loop(int n)
{
    int m;
    scanf("%d", &m);
    {
        int i;
        for(i=m; i<n; i++){
            puts(".....")
        } // 内层块可以使用外层块的m。
    }
    return m;
}

若内层块定义了与外层块同名的变量，则**会隐藏外层块的定义**，离开内层块后，外层块变量的作用域又回到了原来的区域。

```c
#include <stdio.h>
int main()
{
    int x = 30;      // original x
    
    printf("x in outer block: %d at %p\n", x, &x);
    {
        int x = 77;  // new x, hides first x
        printf("x in inner block: %d at %p\n", x, &x);
    }
    printf("x in outer block: %d at %p\n", x, &x);
    while (x++ < 33) // original x
    {
        int x = 100; // new x, hides first x 
        x++;
        printf("x in while loop: %d at %p\n", x, &x);
    }
    printf("x in outer block: %d at %p\n", x, &x);

    return 0;
}
```
程序输出：
```c
x in outer block: 30 at 000000000062FE4C
x in inner block: 77 at 000000000062FE48
x in outer block: 30 at 000000000062FE4C
x in while loop: 101 at 000000000062FE44
x in while loop: 101 at 000000000062FE44
x in while loop: 101 at 000000000062FE44
x in outer block: 34 at 000000000062FE4C
```

### 初始化
**自动变量不会初始化**，除非显式初始化它。

## 02 寄存器变量
|存储类别|存储期|作用域|链接|声明方式|
|---|---|---|---|---|
|寄存器|自动|块|无|块内，使用关键字register|

寄存器变量储存在CPU的寄存器中，或者概括地说，储存在最快的可用内存中。与普通变量相比，访问和处理这些变量的速度更快。

## 03 静态无链接变量
|存储类别|存储期|作用域|链接|声明方式|
|---|---|---|---|---|
|静态无链接变量|静态|块|无|块内，使用关键字static|

静态的意思是**该变量在内存中原地不动**。
使用``static``来声明静态无链接变量。

## 04 静态外部链接变量
|存储类别|存储期|作用域|链接|声明方式|
|---|---|---|---|---|
|静态外部链接|静态|文件|外部|所有函数外|

注意⚠️：如果一个源代码文件使用的外部变量定义在另一个源代码文件中，则必须使用extern在该文件中声明该变量。
```c
int Errupt; // 外部定义的变量
double Up[100]; // 外部定义的数组
extern char Coal; // 如果Coal被定义在另一个文件中，则必须这样声明

void next(void);
int main()
{
    extern int Errupt; // 可选的声明
    extern double Up[]; //可选的声明
    ...
}
void next(void)
{
    ...
}
```

>[!warning]
>+ 在main()中声明Up数组时（这是可选的声明）不用指明数组大小，因为第一次声明已经提供了数组的大小信息。
>+ main中的两条extern声明完全可以省略，因为外部变量具有**文件作用域**。

>[!important]
>+ 静态外部链接变量与自动变量不同的是，如果未初始化静态外部链接变量，它们会被**自动初始化为0**。这一原则也适用于外部定义的数组元素。
>+ 与自动变量的情况不同，只能使用**常量表达式初始化文件作用域变量**。
```c
int x = 10;
int y = 3 + 20;
size_t = sizeof(int);
int x2 = 2*x; // 不行，因为x是变量。
```
**只要不是变长数组，sizeof表达式可视为常量表达式。**

## 05 静态内部链接变量
|存储类别|存储期|作用域|链接|声明方式|
|---|---|---|---|---|
|静态内部链接|静态|文件|内部|所有函数外，使用关键字static|

```c
#include <stdio.h>
static int sum = 0; //sum变量只能在本文件中引用。
int main()
{
    int a,b;
    extern int sum; // 可选的重复声明。
    scanf("%d%d", &a, &b);
    sum += a+b;
    printf("%d", sum);

    return 0;
}
```

## 06 定义和声明
```c
int tern = 1;
main()
{
    extern int tern;
}
```
这里tern被声明了两次。第一次声明为变量预留了存储空间，第二次只是告诉编译器使用之前已经创建的tern变量。

因此，第一次声明称为**定义式声明**，第二次声明称为**引用式声明**。

## 07 存储说明符
### （1）static
对于自动变量，改变其存储期为静态存储期。对于外部变量，改变其链接为内部链接。

### （2）extern
对于变量，只作用于外部变量（文件作用域）。
