# 12 - 1 结构变量

## 01 建立结构声明
结构声明描述了一个结构的组织布局。如下：
```c
struct book{
    char title[100];
    char author[100];
    float value;
};
```
该声明并未创建实际的数据对象，只描述了该对象由什么组成。在结构声明中，用一对花括号括起来的是**结构成员列表**。成员可以是任意一种C语言的数据类型，也可以是其他的结构。

## 02 定义结构变量
**结构**有两层含义，一层含义是“结构布局”，结构布局告诉编译器如何表示数据，但是**它并未让编译器为数据分配空间**。下一步是创建一个结构变量，即是结构的另一层含义。
```c
struct book library;
```
这行代码创建了一个结构变量library，编译器使用book模板为该变量分配空间。

在结构变量的声明中，``struct book``相当于一般声明中的int、float等，因此，可以定义两个struct类型的变量，或者是指向struct book类型的指针。

定义结构变量通常有三种形式：
```c
struct book{
    ...
} library;

struct book library;

struct {
    ...
} library; // 无结构标记，该模板为一次性的。
```

## 03 初始化结构
```c
struct book library = {
    "The Pious Private...",
    "Renee Vivotte", 
    1.95
};
```

>[!warning]
>初始化静态存储期的变量必须使用**常量值**，或者**常量表达式**，这对初始化静态存储期的结构同样适用。

### C99：指定初始化器
C99和C11为结构提供了指定初始化器，其语法与数组的指定初始化器类似。
```c
struct book gift = {.value = 25.99, .author = "James Broadfool", .title = "Rue for the Toad"};
```

## 04 访问结构成员
使用结构成员运算符——点(.)访问结构中的成员。例如，library.value即访问library的value部分。

## 05 嵌套结构
有时，在一个结构中包含另一个结构（即嵌套结构）很方便。
```c
struct birthday{
    int year;
    int month;
    int day;
};
struct student{
    char name[40];
    struct birthday brith;
    int age;
};
struct student stu = {"James", {2000, 10, 24}, 25};
```

## 06 设置指向结构的指针
```c
struct student * ptr;
ptr = &stu;
```

>[!warning]
>和数组不同，结构的名称并不是首元素的地址，因此要加取地址符。

## 07 用指针访问成员
### （1）使用 -> 运算符
上例中，
```c
ptr -> name   //相当于 stu.name
ptr -> birth  //相当于 stu.birth
ptr -> age    //相当于 stu.age
```

### （2） 使用解引用运算符
上例中，
```c
(*ptr).name   //相当于stu.name
(*ptr).birth  //相当于stu.birth
(*ptr).age    //相当于stu.age
```

>[!warning]
>这里要注意，必须使用()小括号，因为 运算符 . 的优先级比解引用 * 的优先级高。
