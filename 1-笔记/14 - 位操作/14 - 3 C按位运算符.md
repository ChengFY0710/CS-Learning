# 14 - 3 C按位运算符
按位运算符**不会改变**原变量的值。

## 01 按位逻辑运算符
4个按位逻辑运算符都应用于**整型数据**，包括char类型。

### （1）二进制反码或按位取反 ~
一元运算符~把 1 变为 0 ，把 0 变为 1 。例如 ~(1000110) == 0111001 。
>[!note]
>改运算符不会改变变量的值。

### （2）按位与 &
二元运算符，通过**逐位比较**两个运算对象得出一个新值。若两个运算对象**相同位置的位**上数字**同时为1（真）**，结果中**对应位数**的数字就为1。
```c
val1         = 10010001011
val2         = 10010110110
val1 & val2 == 10010000010
val1 &= val2;
val1        == 10010000010
```

### （3）按位或 |
二元运算符，通过**逐位比较**两个运算对象得出一个新值。若两个运算对象**相同位置的位**上数字**有一个为1（真）**，结果中**对应位数**的数字就为1。
```c
val1         = 10010001011
val2         = 10010110110
val1 | val2 == 10010111111
val1 |= val2;
val1        == 10010111111
```

### （4）按位异或 ^
二元运算符，通过**逐位比较**两个运算对象得出一个新值。若两个运算对象**相同位置的位**上数字**不相同**，结果中**对应位数**的数字就为1。
```c
val1         = 10010001011
val2         = 10010110110
val1 ^ val2 == 00000111101
val1 ^= val2;
val1        == 00000111101
```

## 02 移位运算符
### （1）左移 <<
```c
(1001101101) << 2
outcome: (0110110100)
```
按位左移运算符右边的数字，左侧移走的数据丢失，空白处用**0**来填补。

### （2）右移 >>
```c
(1001101101) >> 2
outcome: (0010011011)
```
按位右移运算符右边的数字，右侧移走的数据丢失。
+ 对于无符号类型，空白处用**0**来填补。
+ 对于有符号类型，结果取决于机器。可能用 0 ，也可能用符号位。

>[!note]
>除了 ~，其他都有 &= |= ^= <<= >>=
