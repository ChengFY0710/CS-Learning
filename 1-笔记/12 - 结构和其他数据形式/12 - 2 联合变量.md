# 12 - 2 联合变量

## 01 联合简介
联合（union）是一种数据类型，能在**同一个内存空间中**储存不同的数据类型（不是同时储存）。其典型的用法是，设计一张表以**储存既无规律、事先也不知道顺序的混合类型**。
```c
union hold{
    int digit;
    double bigfl;
    char letter;
};
```
与结构变量声明类似，以上声明可以创造一个可以储存int、double、char类型的值，**但是声明的联合只能储存一个int类型的值或一个double类型的值或一个char类型的值**。
```c
union hold fit;
union hold save[10];
union hold * pu;
```
第一个声明创建了一个单独的联合变量fit，它的大小是**声明中占用最大字节类型的大小**。本例中，占用空间最大的是double类型的数据，一般占8个字节，于是，``sizeof(fit)``就是8。

## 02 联合变量的初始化
可以初始化联合，但要注意的是，联合只能存储一个值。
```c
union hold valB = {88}; // 默认初始化第一个元素。
union hold valC = {.bigfl = 118.2}; // 使用指定的初始化器。
```

## 03 使用联合
```c
fit.digit = 23;
fit.bigfl = 2.0;
fit.letter = 'h';
```
>[!warning]
>在联合中，一次只存一个值，即使有足够的空间，也不能同时存两个值。

和使用指针访问结构使用``->``一样，用指针访问联合时也要使用``->``。
```c
union hold * ptr;
ptr = &fit;
x = ptr->digit //相当于 x = fit.digit
```
