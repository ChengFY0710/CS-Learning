# 12 - 3 枚举类型

使用``enum``关键字，可以创建一个新“类型”并指定它可具有的值。

## 01 建立枚举类型
```c
enum spetrum {red, orange, yellow, green, blue, violet};
enum spetrum color;
```
第一个声明创建了``spetrum``作为标记名，第二个声明使color作为该类型的变量。由于第一个声明中枚举了spetrum类型变量可能有的值，因此，color可能的值是``red, orange, yellow``等。这些符号常量被称为**枚举符**。

## 02 枚举符（enum常量）
enum常量从基础层面看，它们是**int 类型**的常量。

在枚举中，符号常量的**默认值**依次是**0,1,2,3,4,...**，因此只要是能使用整型常量的地方即可以使用枚举常量。

## 03 赋值
在枚举声明中，可以为枚举常量**指定整数值**。
```c
enum levels{
    low = 100, medium = 500, high = 2000
};
```

## 04 enum的用法
```c
enum size{
    red, orange, yellow, green, blue, purple
};
enum size color;
color = red;
```
