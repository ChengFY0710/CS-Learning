# 12 - 5 链表

## 01 创建链表
以创建一个含10个单位的链表。
```c
#include <stdio.h>
#include <stdlib.h>
#define LEN sizeof(int)
struct number{
    int num;
    struct number * next;
};
int main()
{
    struct number * head, * p1, * p2;
    head = p1 = p2 = (struct number *)malloc(LEN);
    for(int i=1; i<10; i++){
        p1 = (struct number *)malloc(LEN);
        p2->next = p1;
        p2 = p1; 
    }
    p2->next = NULL;

    return 0;
}
```

## 02 遍历链表
```c
p1 = p2 = head;
while(p1->next != NULL){
    p1->num = 100;
    p1 = p1->next;
}
```

## 03 释放链表
```c
p1 = p2 = head;
while(p1->next != NULL){
    p2 = p1->next;
    free(p1);
    p1 = p2;
}
```

## 04 链表的删除
假设要删除第4个节点
![12 - 5 - 1](/引用文件/12%20-%205%20-%201.png)
```c
p1 = p2 = head;
for(int i=1; i<3; i++){
    p1 = p1->next;
}
// 此时p1指向第3个节点
p2 = p1->next;  // 使得p2指向第4个节点
p1->next = p2->next; 
```

## 05 链表的插入
假设要把第四个节点插入回去

// 设有第四个节点ptr
p1 = p2 = head;
for(int i=1; i<3; i++){
    p1 = p1->next;
