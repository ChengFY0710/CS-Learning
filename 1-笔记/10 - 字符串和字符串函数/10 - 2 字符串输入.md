# 10 - 2 字符串输入

## 00 分配空间
要把一个字符串读入程序，首先要做的就是**分配空间**。
最简单的方法是，在声明时显示**指明字符数组的大小**。

## 01 gets函数
在读取字符串时，scanf和转换说明%s**只能读取一个单词**。
gets函数**读取整行输入**，直到遇到换行符，然后**丢弃换行符**，存储其余字符，并在末尾添加‘\0’空字符，使其成为一个字符串。

>[!caution]
>gets函数的参数只有数组名，因此它不会检查数组长度是否能够容纳输入字符串的长度，可能会导致**缓冲区溢出**。

## 02 gets的替代品 —— fgets函数
fgets函数有三个参数``fgets(数组名， 长度， 输入来源)``
+ 若长度为n，则fgets会**读取n-1个字符**，或**遇到换行符\n停止读取**。然后**在末尾添加‘\0’空字符**，使其成为一个字符串。
+ fgets读取到换行符**不会丢弃**，而是将其存入字符串中。
+ 输入来源可以是其他文件，也可以是键盘输入**stdin**，``stdin``标识符定义在``<stdio.h>``头文件中。

>[!warning]
>gets函数在C11标准中已被废除，但部分编译器仍然支持gets函数。

### （1）fgets函数的返回值
fgets函数发返回值是一个**char类型的指针**，其含义取决于函数的执行结果：
+ **成功读取时**：返回指向**第一个参数，即接受输入的字符串/数组**的指针（地址）。
+ **遇到文件结束时（EOF）时**：返回**NULL指针（空指针）**，且缓冲区内容可能不完整（未定义），该指针保证不会指向有效数据，所以可以用于表示这种情况。
+ **遇到错误时**：该函数也返回**NULL指针**。

>[!tip]
>Windows系统可以通过键盘Ctrl + Z来传入EOF（文件结束）信号。

>[!note]
>**空字符和空指针**
>相同点：都可以用**数字0**来表示。
>不同点：空字符（或'\0'）本质上是一个字符，是**char类型**，**用来标记字符串末尾的字符**。
>空指针（或NULL）本质上是一个指针，是**指针类型**，它并不指向任何有效数据的地址。

### （2）如何处理掉fgets新增的换行符
+ 可以遍历整个字符串，将\n替换成\0
```c
for(i=0; i<n; i++){
    if(str[i] == '\n')
        str[i] = '\0';
}
```
### （3）如何处理掉fgets没有储存留在缓冲区中的字符？
+ 可以通过循环，来清除缓冲区
```c
while((c=getchar() != '\n'))
    continue;   // 读取但不存储，直到\n（包括\n）
```

### （4）删除换行符，并丢弃数组装不下的字符
```c
#include <stdio.h>
int main()
{
    char str[15];
    int i;
    char c;

    while(fgets(str, 15, stdin) != NULL && str[0] != '\0'){
        i = 0;
        while(str[i] != '\n' && str[i] != '\0')
            i++;                                // 遍历字符串，直到遇到换行符或空字符。
        if(str[i] == '\n')
            str[i] = '\0';
        else{
            while((c=getchar()) != '\n')
                continue;
        }
        puts(str);
    }

    return 0;
}
```
>[!tip]
>fgets函数和gets函数都已在``<stdio.h>``头文件中声明和定义。

## 03 gets_s函数
C11新增的gets_s函数为可选函数，并不是所有编译器都支持。
它的格式如下：
```c
gets_s(words, lenth);
```
与fgets的区别如下：
+ gets_s函数只从标准输入中读取数据。
+ gets_s函数读取到换行符，**会丢弃它**，并停止读取。

## 04 scanf函数
scanf函数最大的不同是如何确定字符串的末尾。
+ scanf函数都是从**第1个非空白字符作为字符串的开始**，以**下一个非空白字符作为字符串的结束**。
+ 如果指定了字符宽度，如``scanf("%15s", &a);``，那么scanf()将读取15个字符或者读到第一个非空字符处停止。

>[!warning]
>scanf函数和gets函数一样，不会检查数组长度是否容得下输入的字符串。不过，可以在%s转换说明中使用**字段宽度**防止溢出。
