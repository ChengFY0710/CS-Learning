# 10 - 4 字符串函数

## 01 字符串连接函数——strcat函数
其一般形式为**strcat(字符数组1, 字符数组2)**

其作用是把两个字符数组中的字符串连接起来，把字符串2接到字符串1后面，**结果放在字符数组1中**。

函数调用后会返回一个函数数值——**字符数组1的地址**。
```c
#include <stdio.h>
#include <string.h>
int main()
{
	char str1[]={"Xiaomi "};
	char str2[]={"HyperOS 3"}; 
	strcat(str1,str2);
	printf("%s",str1);
	
	return 0;
 } 
 ```

>[!warning]
>+ strcat()函数不检查str1的长度，因此字符数组1必须足够大，以便容纳连接后的新字符串。否则会导致**缓冲区溢出**。

```c
#include <stdio.h>
#include <string.h>
int main()
{
	char str1[60] = "Xiaomi HyperOS ";
	char str2[60] = "Harmony OS\0HUAWEI";
	strcat(str1, str2);
	printf("%d %d\n", strlen(str1), strlen(str2));
	puts(str1);
	
	return 0;
}
//str1连接完后，为"Xiaomi HyperOS Harmony OS"
//若反过来, str2为"Harmony OSXiaomi HyperOS "

//因此：
char str3[60] = "Xiaomi HyperOS\0XiaomiHyperCore";
char str4[30] = "Harmony OS\0HUAWEI";
strcat(str3, str4);
//str3为Xiaomi HyperOSHarmony OS\0rCore
```
>[!warning]
>+ strcat()函数在连接时，连接在str1的第一个\0后面，且**从第一个\0处开始覆盖式复制**，并只将str2的**有效字符串段**连接。
>+ strcat()函数的连接实际上是**有效字符串的覆盖式复制**。

>[!note]
>连接前两个字符串的后面都有′\0′，连接时将字符串1后面的′\0′取消，只在新串最后保留′\0′。

## 02 字符串复制函数——strcpy函数
```c
char str1[]="Xiaomi HyperOS 3";
char str2[]="Xiaomi HyperCore";
strcpy(str1, str2);
//or
char str[]="Xiaomi SU7"
strcpy(str1, "Xiaomi YU7");
```
作用：将字符串2复制到字符数组1中去。**该函数返回str1的地址。**
>[!important]
>+ （1）“字符数组1”必须写成**数组名**形式，“字符串2”可以是**字符数组名**，也可以是一个**字符串常量**。
>+ （2）字符串1必须**足够长**以容纳字符串2。
>+ （3）若在复制前未对字符数组1初始化或赋值，则其各字节中的内容**无法预知**，复制时将字符串2和其后的′\0′一起复制到字符数组1中，**取代字符数组1中前面的字符**，未被取代的字符**保持原有内容**。

>[!caution]
>不能直接将字符串赋值给数组，数组名只是一个地址！

### 复制前n个字符——strncpy函数
```c
char str1[10]="Xiaomi";
char str2[10]="HyperOS";
strncpy(str1, str2, 3);
```
strncpy函数的作用是将str2中的前n个字符复制到str1中，**末尾仍保留\0**。

```c
#include <stdio.h>
#include <string.h>
int main()
{
	char str1[]="Xiaomi HyperOS 3";
	char str2[]="HUAWEI";
	strcpy(str1, str2);
	printf("%s", str1);
	
	return 0;
}
// 输出 HUAWEI

#include <stdio.h>
#include <string.h>
int main()
{
	char str1[]="Xiaomi HyperOS 3";
	char str2[]="HUAWEI";
	strncpy(str1, str2, 3);
	printf("%s", str1);
	
	return 0;
}

// 输出 HUAomi HyperOS 3
```

>[!warning]
>strncpy函数和strcpy函数相同，都是**遇到\0字符停止读取**。但不同的是，strcpy是**整体替代式复制**，strncpy是**部分替换式复制**。

## 03 字符串比较函数——strcmp函数
```c
strcmp("Xiaomi", "HUAWEI");
strcmp(str1, str2);
strcmp(str1, "HUAWEI");
```
作用：比较字符串1和字符串2。

规则是：将两个字符串**自左至右逐个字符**相比(按**ASCII码值**大小比较)，直到出现**不同的字符或遇到′\0′为止**。

比较结果：
+ 如果字符串1与字符串2相同，则函数值为0。
+ 如果字符串1>字符串2，则函数值为一个**正整数 1**。
+ 如果字符串1<字符串2，则函数值为一个**负整数 -1**。

### 比较前n个字符strncmp()函数
```c
strncmp(str1, str2, 3);
```
同样读取到\0时结束。该函数在查找以某些字符开头的字符串时很有用。

## 04 字符串长度测量函数——strlen函数
strlen函数可以测量字符串的**有效长度**。

```c
#include <stdio.h>
#include <string.h>
#define NAME Xiaomi HyperOS // 14个字符
int main()
{
	char name[20];
	scanf("%s",name); 
	printf("%zd",strlen(name));
	printf("%zd",strlen(NAME));// NAME中字符串的长度为15（包含\0）
	//输出 14
	return 0;
}
```
>[!note]
>strlen默认返回的是**无符号整数**类型，推荐使用``%u``转换说明符。

## 05 字符串转换大小写函数——strlwr函数和strupr函数
strlwr(字符串或数组名) 将**大写转换为小写**。
strupr(字符串或数组名) 将**小写转换为大写**。

## 06 sprintf()函数
sprintf函数定义在stdio.h中，与printf函数类似，唯一的区别是将内容输出到字符串中。
```c
sprintf(str1, "%s %d", str2, a);
```
第一个参数为写入字符数组的地址。
