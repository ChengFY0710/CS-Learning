# 15 - 2 操作文件的函数

>[!warning]
>本文提供的函数都定义在 stdio.h 中。

## 01 fopen()函数
fopen()函数可用于打开文件，该函数声明在stdio.h中。
```c
fopen(p1, "r");
```
第一个参数是**待打开文件的名称**，更确切地说是一个包含该文件名的字符串**地址**。第二个参数是一个**字符串**，指定打开文件的模式。

![15 - 2 - 1](/引用文件/15%20-%202%20-%201.png)

>[!warning]
>如果使用任何一种“w”模式打开现有的一个文件，该文件的内容会被**全部删除**，以便程序在一个空白的文件中开始操作。

程序成功打开文件后，fopen()将返回**文件指针**。

## 02 getc() 和 putc() 函数
```c
ch = getchar();
ch = getc(fp);
```
简单来说，就是**从文件中**获取**一个字符**，**向文件中**输出**一个字符**。

## 03 fclose() 函数
```c
fclose(fp);
```
fclose()函数关闭fp指定的文件，必要时刷新缓冲区。
若**成功关闭文件**，fclose函数返回**0**，否则返回**EOF**。
```c
if(fclose(fp) != 0)
    printf("Error in closing file %s\n", argv[1]);
```

## 04 fprintf()和fscanf()函数
```c
int main()
{
    FILE * fp1, * fp2;
    char words[100];
    fp1 = fopen("text", "w");
    fprintf(fp1, "This is an example");
    fclose(fp1);
    fp2 = fopen("text", "r");
    fscanf(fp2, "%s", words);
    fclose(fp2);
}
```
## 05 fgets()和fputs()函数
```c
int main()
{
    FILE * fp1, * fp2;
    char words[100];
    fp1 = fopen("text", "w");
    fputs("Xiaomi HyperOS", fp1);
    fclose(fp1);
    fp2 = fopen("text", "r");
    fgets(words, 101, fp2);
    fclose(fp2);
}
```

## 06 随机访问：fseek()函数
有了fseek()函数，便可把文件看作是数组，在fopen打开的文件中**直接移动到任意字节处**。
```c
fseek(fp, -5L, SEEK_SET);
```

+ fseek()的第1个参数是**FILE指针**，指向待查找的文件，fopen()应该**已打开该文件**。
+ fseek()的第2个参数是**偏移量**，该参数表示**从起始点开始要移动的距离**。该参数必须是一个**long类型**的值，可以为**正（前移）、负（后移）或0（保持不动）**
+ fseek()的第3个参数是**模式**，该参数确定**起始点**。

根据ANSI标准，在stdio.h头文件中规定了几个表示模式的明示常量：
|模式|偏移量的起始点|
|:---:|:---:|
|SEEK_SET|文件开始处|
|SEEK_CUR|当前位置|
|SEEK_END|文件末尾|

旧的实现可能缺少这些定义，可以使用数值**0L、1L、2L**分别表示这三种模式。L表示long类型。

如果一切正常，fseek()的返回值为**0**；如果出现错误，其返回值为**负一**。

## 07 随机访问：ftell()函数
ftell()函数的返回类型是**long**，它返回的是**当前的位置**。此外，ftell函数的返回值是**long类型**。

## 08 fread()函数
fread 和 fwrite 是**用于二进制文件读写的函数**，它们**以块为单位**进行数据传输，适用于处理**结构体、数组**等非文本数据。
```c
size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream);
```
size_t是**无符号整型**。
+ 第一个参数是**指向存储读取数据的内存地址**。
+ 第二个参数是**每个数据项的大小（以字节为单位）**。
+ 第三个参数是**要读取的数据项的数量**。
+ 第四个参数是**文件指针，指向源文件**。

fread()的作用就是从**二进制文件**中读取某一**以块为单位**的**结构体、数组等数据**，写入到ptr中。

返回值：**成功时**返回实际**读取的数据项数量**；失败时返回 0。

## 09 fwrite()函数
```c
size_t fwrite(const void *ptr, size_t size, size_t nmemb, FILE *stream);
```
+ 第一个参数是**指向要写入数据的内存地址**。
+ 第二个参数是**每个数据项的大小（以字节为单位）**。
+ 第三个参数是**要读取的数据项的数量**。
+ 第四个参数是**文件指针，指向源文件**。

fwrite()的作用就是将**以块为单位的**（例如**结构体，数组**等）数据写入到**二进制文件**中。

返回值：**成功时**返回实际**写入的数据项数量**；失败时返回 0。

```c
#include <stdio.h>

int main() {
    FILE *fp;
    int data[3] = {1, 2, 3};
    int read_data[3];

    // 写入数据
    fp = fopen("data.bin", "wb");
    if (fp == NULL) {
        return 1;
    }
    fwrite(data, sizeof(int), 3, fp);
    fclose(fp);

    // 读取数据
    fp = fopen("data.bin", "rb");
    if (fp == NULL) {
        return 1;
    }
    fread(read_data, sizeof(int), 3, fp);
    fclose(fp);

    // 输出读取的数据
    for (int i = 0; i < 3; i++) {
        printf("%d ", read_data[i]);
    }

    return 0;
}
```
