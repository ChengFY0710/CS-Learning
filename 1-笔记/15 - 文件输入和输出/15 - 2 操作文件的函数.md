# 15 - 2 操作文件的函数

## 01 fopen()函数
fopen()函数可用于打开文件，该函数声明在stdio.h中。
```c
fopen(p1, "r");
```
第一个参数是**待打开文件的名称**，更确切地说是一个包含该文件名的字符串**地址**。第二个参数是一个**字符串**，指定打开文件的模式。

![15 - 2 - 1](/引用文件/15%20-%202%20-%201.png)

>[!warning]
>如果使用任何一种“w”模式打开现有的一个文件，该文件的内容会被**全部删除**，以便程序在一个空白的文件中开始操作。

程序成功打开文件后，fopen()将返回**文件指针**。

### 文件指针
```c
int main()
{
    FILE * fp;
    fp = fopen(text, "r");
}
```
FILE 是定义在stdio.h中的派生类型。fp并不指向实际的文件，它指向一个包含文件信息的数据对象。
